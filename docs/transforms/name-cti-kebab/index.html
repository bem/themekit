name/cti/kebab

Create a kebab-case name from path of tokens.

note

All notations for example camelCase will be transformed in kebab-case too.

Usage#

Configure tool:

themekit.config.json
{
"entry": {
"default": "./themes/default.theme.json"
},
"output": {
"css": {
"transforms": ["name/cti/kebab"],
"buildPath": "./themes",
"files": [
{
"destination": "[entry]/root.css",
"format": "css/variables"
}
]
}
}
}

Configure theme:

themes/default.theme.json
{
"sources": ["./themes/tokens/*.tokens.yml"]
}

Example#

  • tokens/yaml
  • css
color:
controls:
typeAaction:
typoColor:
base:
value: "#000"
fillColor:
base:
value: "#fff"
